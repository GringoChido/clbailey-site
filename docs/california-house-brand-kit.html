04 / Spacing System
Vertical Rhythm & Section Padding
The site uses generous whitespace with consistent vertical padding on sections. The base unit is approximately 10px, scaling up in multiples.

Container Widths

Full bleed sections use 100vw with no max-width
05 / Layout Patterns
Page Structure
The site follows a clean, section-based vertical layout. Full-width image sections alternate with contained content sections. Navigation uses a semi-transparent overlay bar.

Key Layout Rules
1. Full-width hero images with centered overlay text
2. Sections alternate between white, #f9f9fb, and #ebecee backgrounds
3. Product grids use equal-ratio cells (square or 4:3)
4. Category labels are always Raleway uppercase centered above content
5. Navigation is fixed/sticky with semi-transparent background
6. Footer uses the darkest color (#1b1b1b) for contrast
Grid System
Product catalog: 3-column grid, no gap (edge-to-edge)
Content sections: Single column, centered, max-width 1100px
Feature rows: 2-column (image + text), 50/50 split
Category navigation: 3 or 4 column even grid with image backgrounds

07 / Design Principles
Style Summary
Do
Use light font weights (300) for large display text
Keep color palette muted and tonal (grays only)
Use generous whitespace between sections
Uppercase + letter-spacing for labels and navigation
Let photography do the heavy lifting for visual interest
Keep UI elements minimal and understated
Use semi-transparent overlays on images
Maintain high contrast between sections via background shifts
Don't
Use bright or saturated accent colors
Use heavy font weights for headings (keep it 300-400)
Add drop shadows or heavy border treatments
Use rounded corners on cards or buttons
Crowd content with tight spacing
Use more than 2 font families in one section
Add decorative borders or ornaments
Make buttons visually dominant (they should be subtle)
